Клонирование проекта в папку nameProject:
git clone https://... nameProject


Добавление файлов в commit:
git add .


Запись commit`а:
git commit -m "Комментарий"


Отправить commit:
git push


Принять изменения:
git pull


Авторство коммитов:
git config --global user.email "Ваш email"
git config --global user.name "Ваше Имя"
